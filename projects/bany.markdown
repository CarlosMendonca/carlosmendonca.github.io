# Bany
{: .fs-9 }

[View it on GitHub](https://github.com/CarlosMendonca/bany){: .btn .fs-5 .mb-4 .mb-md-0 }

Bany is an envelope budgeting web application with multi-currency support, YNAB import capability, and full-text search. It lets you organize spending into envelopes, manage budgets across multiple currencies, and migrate existing data from YNAB. The app is built with Elixir and the Phoenix 1.8 framework on a PostgreSQL backend, with a real-time interface powered by Phoenix LiveView 1.0 and Tailwind CSS.
