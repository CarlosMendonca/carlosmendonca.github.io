# Juntos
{: .fs-9 }

[View it on GitHub](https://github.com/CarlosMendonca/juntos){: .btn .fs-5 .mb-4 .mb-md-0 }

Juntos is a cross-platform Flutter app for shared expense tracking and trip planning. Organize spending into groups, split costs flexibly, monitor running balances, and settle up with other members. Built on Firebase (Auth + Firestore), Riverpod, and GoRouter.
