# Juntos
{: .fs-9 }

[View it on GitHub](https://github.com/CarlosMendonca/juntos){: .btn .fs-5 .mb-4 .mb-md-0 }

Juntos is a multi-platform Flutter application for expense sharing and trip planning. It lets you track expenses, split costs flexibly across participants, view running balances, organize spending into groups, and settle payments between members. The app is built with Flutter for cross-platform support, Firebase for authentication and real-time data via Firestore, Riverpod for state management, and GoRouter for navigation.
