# Nix Flakes
{: .fs-9 }

A collection of Nix flakes I maintain to package software not yet available in nixpkgs.
{: .fs-6 .fw-300 }

- **[cliamp-flake](https://github.com/CarlosMendonca/cliamp-flake)** — A Nix flake for cliamp, a terminal music player by Bjarne Øverli.
- **[git-fourchette-flake](https://github.com/CarlosMendonca/git-fourchette-flake)** — A Nix flake for gitfourchette, a comfortable Git UI by Iliyas Jorio.
- **[pocket-sync-flake](https://github.com/CarlosMendonca/pocket-sync-flake)** — A Nix flake for pocket-sync, a sync tool for the Analogue Pocket by neil-morrison44.
